# ImageCrop
ImageCrop 主要用于剪切大图获取小图！


# How to
  ## Step 1. Add it in your root build.gradle at the end of repositories:
    allprojects {
      repositories {
        ...
        maven { url 'https://jitpack.io' }
      }
    }

  ## Step 2. Add the dependency
    dependencies {
            implementation 'com.github.sinothk:ImageCrop:1.0.0409'
    }
